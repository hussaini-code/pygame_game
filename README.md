# Tic Tac Toe Game 

A simple and beautiful Tic Tac Toe game with two_player fuctionality

## Features
- Responsive user interface
- Automatic winner detection
- Restart game capability

## Technological Used
- python
- pygame
- numpy

## How to Run 
1. Make sure Python 3.11 is installed on your system.
2. install pygame, install numpy 
command : pip install pygame/ numpy
3. Run the game: python main.py
